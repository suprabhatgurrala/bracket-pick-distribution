# Bracket Pick Distribution

When filling out a March Madness bracket, knowing how many other people picked a certain team to win a certain round can be helpful in making your picks. This app displays the pick distribution for each team in the tournament.

This information used to be published by ESPN on their website, but they stopped doing so in 2025. This app recreates that functionality.
Yahoo still publishes this information on their website: https://tournament.fantasysports.yahoo.com/mens-basketball-bracket/pickdistribution

## Running Locally
```
uv sync
uv run streamlit run main.py
```
